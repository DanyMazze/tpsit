HTML e CSS

HTML (HyperText Markup Language) e CSS (Cascading Style Sheets) sono i pilastri del web. HTML definisce la struttura e il contenuto di una pagina, mentre CSS si occupa dello stile e dell'aspetto visivo, come colori, font e layout. Bootstrap

Bootstrap è un framework CSS gratuito e open-source progettato per creare siti web e applicazioni responsive in modo rapido. Fornisce una vasta gamma di componenti predefiniti come pulsanti, modali, griglie e menu di navigazione, oltre a classi pronte per personalizzare facilmente il design. JavaScript

JavaScript (JS) è un linguaggio di programmazione utilizzato per rendere le pagine web interattive e dinamiche. Permette di aggiungere funzionalità come aggiornamenti in tempo reale, animazioni, validazione dei moduli e interazioni con l'utente.
