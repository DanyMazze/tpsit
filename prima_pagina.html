  • Che cose il WEB?

    • Il Web è uno spazio elettronico e digitale di Internet destinato alla pubblicazione di contenuti multimediali

  • Che problemi risolveva il WEB ?

    • Il Web ha risolto il problema dell'accesso limitato e frammentato alle informazioni, rendendole disponibili globalmente e in modo immediato tramite la rete Internet.

  • Che differenza tra web e html ed internet?

    • Internet è la rete globale di computer interconnessi che consente lo scambio di dati e comunicazioni. Web è il sistema di documenti e pagine interconnessi che utilizziamo per navigare su Internet tramite browser. HTML è il linguaggio di markup utilizzato per creare e strutturare le pagine web visualizzabili sul Web.
